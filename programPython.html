from random import randint
def genere_alea(n): 
    liste = []
    for i in range(1000): 
        liste.append(randint(1, n))
    return liste
def occurrences(valeur, tableau): 
    compteur = 0
    for val in tableau : 
    if  (valeur == val): 
            compteur += 1
    return compteur
def minmaxtableau(t): 
    mini = t[0]
    maxi = t[0]
    for valeur in t    : 
    if  (valeur < mini): 
            mini = valeur
    if (valeur> maxi): 
        maxi = valeur
    return (mini, maxi)

ALEA renvoie un nombre réel aléatoire distribué de manière symétrique supérieur ou égal à 0 et inférieur à 1. Puis la fonction genere_alea(n) crée une liste de 1000 nombres aléatoire compris entre 1 et n.


La fonction occurrences (valeur,tableau) retourne le nombre d’occurrences de la valeur dans le tableau.


La fonction minmaxtableau(t) qui contient deux listes min et max qui permette de savoir si la valeur envoyée en paramètre dans le tableau est inférieur au minimum ou supérieur au maximum.


La fonction simulation(), qui génère un tableau de 1000 entiers entre 1 et 10, et affiche ensuite le nombre de fois que chaque nombre a été tiré.
